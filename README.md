# Houselog

A simple tracking system for house-related tasks

Includes:
- frequency tracker
- email reminder (WIP)
- notes (like furnace filter size, or fridge model, etc)

![login](docs/images/login.png)

![dashboard](docs/images/dashboard.png)

### Usage

```
mv .env.example .env
vim .env
docker-compose build
docker-compose up
```
