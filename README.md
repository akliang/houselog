# Houselog

A simple tracking system for house-related tasks

Includes:
- frequency tracker
- email reminder (WIP)
- notes (like furnace filter size, or fridge model, etc)

<img src="docs/images/dashboard.png" width=400>

### Usage

```
mv .env.example .env
vim .env
docker-compose build
docker-compose up
```
